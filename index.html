jquery-scrollbars by Collin Reisdorf

Unobtrusive designer scrollbars. Works with custom css, and completely mimic standard scrollable divs (even native mouse wheel support).

Dependencies

Requires jQuery, the jQuery UI widget factory, and the scrollStart/Stop custom events from http://james.padolsey.com/javascript/special-scroll-events-for-jquery/.

Install

Include jQuery, the jQuery UI widget factory (or more of the UI library) and the scrollStart/Stop plugin, then include the

Example

HTML

<div class="scrollable">
  <p>A bunch of content that will force the containing div to scroll (if 
  it's overflow was set to "auto", (or "scroll")).</p>
  <p>Can be any combination of elements, or I think even raw text, though 
  I haven't tested that yet.</p>
  <p>And if your content doesn't need to scroll not harm, you just 
  won't see anything change (also the scrollbars won't be displayed, (event 
  if you set the scrollable's overflow to "scroll"))</p>
</div>
